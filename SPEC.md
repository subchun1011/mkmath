# 프로젝트명: 키즈 런닝 어드벤처 (Kids Learning Adventure)

## 1. 개요
- **대상**: 6세 ~ 초등 저학년
- **목적**: 반복 학습(산수, 한자, 영어, 시계)을 통한 학업 성취도 향상 및 게임형 UI를 통한 동기 부여.

## 2. 화면 구조 (웹뷰 최적화)
- 전체 높이를 100%로 기준하여 4분할 레이아웃 적용:
  1. **Timer Area (5%)**: 상단 프로그레스 바 형태의 남은 시간 표시.
  2. **Action Area (30%)**: 로봇/자동차 파츠 애니메이션 및 전투 액션 노출.
  3. **Question Area (30%)**: 중앙에 학습 문제 노출 (큰 폰트).
  4. **Input Area (35%)**: 하단 정답 입력 숫자 패드 및 확인 버튼.

## 3. 핵심 기능
- **난이도 시스템**: Level 1 ~ 10에 따른 문제 범위 자동 조절.
- **스테이지 맵**: 슈퍼마리오3 스타일의 경로 기반 스테이지 선택 UI.
- **파츠 시스템**: 정답 시 캐릭터의 특정 파츠(머리, 몸통 등) 업그레이드 및 액션 실행.

## 4. 기술 스택
- **Frontend**: React.js, CSS Modules (애니메이션 포함).
- **Backend**: Node.js (Express), JSON 기반 데이터 관리.

## 5. 문제 엔진 사양
- 모든 문제 생성기는 `(level) => { question, answer, options, hint }` 형식을 따른다.
- 새로운 학습 과목 추가 시 `generators` 폴더에 새 파일을 만들고 `QuestionFactory`에 등록만 하면 즉시 반영된다.
